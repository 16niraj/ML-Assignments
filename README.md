# ES 335 - Machine Learning Assignments

This repository contains all my course assignments for the **Machine Learning** course at *IIT Gandhinagar*.
It covers fundamental and advanced ML concepts implemented through hands-on coding exercises and projects.

---

## Course Overview

The course focused on:

* Supervised Learning (Regression, Classification)
* Unsupervised Learning (Clustering, Dimensionality Reduction)
* Model Evaluation & Validation
* Optimization Techniques
* Decision Trees
* Deep Learning
* Reinforcement Learning
